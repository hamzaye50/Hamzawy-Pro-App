# حمزاوي برو - تطبيق تحميل فيديوهات متكامل
# Hamzawy Pro - Advanced Video Downloader & Social Platform

تطبيق متطور لتحميل الفيديوهات من يوتيوب؏ إنستقرام؏ تيك توك؏ ومنصة X (تويتر) مع ميزات اجتماعية ونظام مكافآت (نقاط حمزاوي).

## الميزات
- **تحميل نظيف**: إزالة العلامة المائية تلقائياً من تيك توك وإنستقرام.
- **اكتشاف المحتوى**: البحث عن الفيديوهات الرائجة واكتشافها عبر المنصات.
- **نظام النقاط**: اكسب نقاطاً مقابل كل تحميل؏ إعجاب؏ أو تعليق.
- **وصف تلقائي**: كتابة وصف عربي جذاب للفيديوهات المحملة بشضة واحدة.
- **التفاعل الاجتماعي**: تابر المستخدمين؏ علّق؏ وأعجب بالفيديوهات.

## التشغيل
1. تثبيت المتطلبات: `pip install -r requirements.txt`
2. تشغيل التطبيق: `python app.py`
3. الوصول للتطبيق: `http://localhost:5000`
